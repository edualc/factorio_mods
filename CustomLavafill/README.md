# CustomLavafill `2.1.0`

Allows the placement of lava just like landfill.

**Original mod:** [Lavafill](https://mods.factorio.com/mod/lavafill) by **Junsung Cho** ([source](https://github.com/junsung-cho/lavafill))

## Changes from original

- Ported to Factorio 2.1: bumped `factorio_version` and dependency floors from `2.0`/`base >= 2.0.0`/`space-age >= 2.0.0` to `2.1`/`base >= 2.1`/`space-age >= 2.1`. No prototype code changes were needed — `place_as_tile.condition` is a valid, unchanged field in the 2.1 API.
